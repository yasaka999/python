# PMO 项目管理系统 - 功能列表

> 最后更新：2026-03-08

---

## 一、系统概述

PMO 项目管理系统是一个面向项目管理办公室（PMO）的轻量级 Web 应用，用于跟踪项目全生命周期、管理里程碑、记录问题风险、统计工时投入，并生成各类报告。

**访问地址：**
- 前端界面：http://142.171.178.36:9000
- 后端 API：http://142.171.178.36:8000
