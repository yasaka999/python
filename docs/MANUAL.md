## 3. 项目管理

### 3.1 查看项目列表

1. 点击左侧菜单「项目列表」
2. 可使用筛选条件：
   - **状态筛选**：可多选（正常、预警、延期等）
   - **阶段筛选**：可多选（启动、实施、验收等）
3. 点击「重置」清空筛选条件

### 3.2 创建新项目

1. 点击「新建项目」按钮
2. 填写项目信息：

| 字段 | 必填 | 说明 |
|------|:----:|------|
| 项目编号 | ✅ | 唯一标识，如 PMO-2024-001 |
| 项目名称 | ✅ | 项目全称 |
| 客户/甲方 | | 客户名称 |
| 项目经理 | | 负责人姓名 |
| 项目阶段 | | 售前/启动/实施/验收/收尾 |
| 项目状态 | | 正常/预警/延期/暂停/已完成 |
| 计划开始日期 | | 格式：YYYY-MM-DD |
| 计划结束日期 | | 格式：YYYY-MM-DD |
| 预算人天 | | 预计投入人天数 |
| 合同编号 | | 合同编号 |
| 区域 | | 项目所在区域 |
| 计划交付日期 | | - |
| 计划初验日期 | | - |
| 计划终验日期 | | - |
| 项目描述 | | 项目简介 |

3. 点击「保存」完成创建

### 3.3 编辑项目

1. 在项目列表中点击项目名称进入详情
2. 点击「编辑」按钮
3. 修改需要更新的字段
4. 点击「保存」

**权限说明：**
- admin/pmo：可编辑所有项目
- member：只能编辑自己创建的项目

### 3.4 删除项目

1. 进入项目详情页
2. 点击「删除」按钮
3. 确认删除

**注意：** 删除项目会同时删除关联的里程碑、问题、风险、人天等数据，请谨慎操作。

### 3.5 项目详情页

项目详情页展示完整信息，并提供快捷入口：

- **里程碑与进度**：管理里程碑节点
- **问题台账**：记录项目问题
- **风险台账**：登记项目风险
- **人天管理**：统计工时投入
- **周报进展**：记录周报
- **报告生成**：下载各类报告

## 4. 里程碑与任务

### 4.1 查看里程碑

1. 进入项目详情页
2. 点击「里程碑与进度」
3. 查看里程碑列表和甘特图

### 4.2 添加里程碑

1. 点击「添加里程碑」
2. 填写信息：

| 字段 | 说明 |
|------|------|
| 名称 | 里程碑名称，如"需求分析完成" |
| 计划日期 | 计划完成日期 |
| 实际日期 | 实际完成日期（可选） |
| 状态 | 未开始/进行中/已完成/延期 |
| 排序 | 显示顺序（数字越小越靠前） |

3. 点击「保存」

### 4.3 更新里程碑状态

1. 找到需要更新的里程碑
2. 点击「编辑」
3. 修改状态和实际日期
4. 保存

### 4.4 任务管理

在里程碑页面下方可管理任务：

1. 点击「添加任务」
2. 填写任务信息：标题、负责人、状态、优先级、截止日期
3. 保存

### 4.5 甘特图

页面右侧展示甘特图视图：
- 蓝色：计划进度
- 绿色：已完成
- 红色：延期

## 5. 问题台账

### 5.1 查看问题列表

1. 进入项目详情页
2. 点击「问题台账」
3. 可按状态、严重等级筛选

### 5.2 登记新问题

1. 点击「添加问题」
2. 填写问题信息：

| 字段 | 说明 |
|------|------|
| 问题标题 | 简明扼要的问题描述 |
| 问题描述 | 详细说明 |
| 严重等级 | 高/中/低 |
| 来源 | 客户/内部/第三方 |
| 负责人 | 问题处理人 |
| 提出日期 | 问题发现日期 |
| 期望解决日期 | 期望完成日期 |
| 状态 | 待处理/处理中/已关闭 |

3. 点击「保存」

### 5.3 更新问题状态

1. 找到需要更新的问题
2. 点击「编辑」
3. 更新状态、解决措施、实际解决日期
4. 保存

### 5.4 关闭问题

当问题解决后：
1. 编辑问题
2. 将状态改为「已关闭」
3. 填写「解决措施」和「实际解决日期」
4. 保存

## 6. 风险台账

### 6.1 查看风险列表

1. 进入项目详情页
2. 点击「风险台账」
3. 可按状态筛选

### 6.2 登记新风险

1. 点击「添加风险」
2. 填写风险信息：

| 字段 | 说明 |
|------|------|
| 风险标题 | 简明扼要的风险描述 |
| 风险描述 | 详细说明 |
| 概率 | 高/中/低 |
| 影响 | 高/中/低 |
| 应对措施 | 风险应对方案 |
| 负责人 | 风险负责人 |
| 状态 | 开放/进行中/已缓解/已关闭 |

3. **风险等级自动计算**：系统根据概率×影响矩阵自动计算
4. 点击「保存」

### 6.3 风险矩阵

| 概率\影响 | 高 | 中 | 低 |
|----------|:---:|:---:|:---:|
| 高 | 🔴高 | 🔴高 | 🟡中 |
| 中 | 🔴高 | 🟡中 | 🟢低 |
| 低 | 🟡中 | 🟢低 | 🟢低 |

### 6.4 更新风险状态

1. 找到需要更新的风险
2. 点击「编辑」
3. 更新状态、应对措施
4. 保存

### 6.5 关闭风险

当风险消除或缓解后：
1. 编辑风险
2. 将状态改为「已关闭」或「已缓解」
3. 保存

## 9. 报告生成

### 9.1 项目级报告

进入项目详情页，点击「报告生成」：

| 报告类型 | 格式 | 说明 |
|----------|------|------|
| 项目周报 | Word | 单项目周度进展报告 |
| 项目月报 | Word | 单项目月度总结报告 |
| 问题风险台账 | Excel | 问题风险明细表 |
| 人天统计 | Excel | 人天投入明细表 |

**操作步骤：**
1. 选择报告类型
2. 选择日期（周报选择日期，月报选择年月）
3. 点击「下载」

### 9.2 整体报告（PMO）

点击左侧菜单「整体报告」：

| 报告类型 | 格式 | 说明 |
|----------|------|------|
| 项目组合综合报告 | Word | 包含执行摘要、重点关注、全局风险问题、里程碑跟踪、状态一览 |
| 项目组合数据汇总 | Excel | 4个Sheet：状态一览、高危问题、高影响风险、逾期里程碑 |

**操作步骤：**
1. 选择报告日期
2. 点击「下载 Word」或「下载 Excel」

### 9.3 报告内容说明

**项目组合综合报告（Word）包含：**

1. **执行摘要**
   - 项目总数统计
   - 按状态分布
   - 按阶段分布

2. **重点关注项目**
   - 延期项目
   - 预警项目

3. **高优先级问题**
   - 严重等级为"高"的开放问题

4. **高影响风险**
   - 风险等级为"高"的开放风险

5. **逾期里程碑**
   - 计划日期已过但未完成的里程碑

6. **项目状态一览表**
   - 所有项目基本信息

## 12. 常见问题

### Q1: 登录失败怎么办？

**可能原因：**
1. 用户名或密码错误
2. 账号被禁用
3. 网络问题

**解决方法：**
1. 检查用户名和密码是否正确
2. 联系管理员确认账号状态
3. 检查网络连接

### Q2: 为什么看不到某些项目？

**可能原因：**
- 您的角色是 member，只能看到自己创建的项目

**解决方法：**
- 联系管理员提升权限为 pmo 或 admin

### Q3: 为什么无法编辑项目？

**可能原因：**
- 您的角色是 member，只能编辑自己创建的项目

**解决方法：**
- 联系项目创建者或管理员

### Q4: 风险等级显示为英文代码怎么办？

**可能原因：**
- 数据库中缺少 risk_level 字典项

**解决方法：**
1. 以管理员登录
2. 进入「系统配置」
3. 检查 risk_level 分类是否有字典项
4. 如缺失，添加以下字典项：
   - rl_h → 高（红色）
   - rl_m → 中（黄色）
   - rl_l → 低（绿色）

### Q5: 报告下载失败怎么办？

**可能原因：**
1. 网络问题
2. 项目数据为空
3. 浏览器拦截下载

**解决方法：**
1. 检查网络连接
2. 确认项目有相关数据
3. 允许浏览器下载弹窗

### Q6: 如何批量导入项目？

目前系统不支持批量导入，需要逐个创建项目。

### Q7: 数据如何备份？

系统每日凌晨 2 点自动备份数据库文件。如需手动备份，请联系管理员。

### Q8: 忘记密码怎么办？

请联系管理员重置密码。
