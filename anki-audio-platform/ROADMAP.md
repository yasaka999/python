# AnkiAudio Platform 开发规划

## 版本规划

### v1.1 - 本地文件支持 (优先级: 高)

**目标**: 支持上传本地音频+字幕文件制卡

**功能点**:
- [ ] 上传音频文件 (mp3, m4a, wav, ogg)
- [ ] 上传字幕文件 (vtt, srt, lrc)
- [ ] 音频+字幕匹配校验
- [ ] 处理流程复用现有 pipeline

**技术方案**:
```
POST /upload
  - multipart/form-data
  - audio_file: 音频文件
  - subtitle_file: 字幕文件 (可选)
  - lang: 语言代码
  - title: 牌组标题
```

**工作量**: 2-3天

---

### v1.2 - 牌组管理 (优先级: 高)

**目标**: 完善牌组管理功能

**功能点**:
- [ ] 删除牌组 (同时删除关联音频文件)
- [ ] 重命名牌组
- [ ] 牌组详情页 (查看所有卡片)
- [ ] 卡片预览 (播放音频+显示字幕)
- [ ] 牌组统计 (卡片数、总时长)

**技术方案**:
```
DELETE /deck/{deck_id}
PATCH /deck/{deck_id} (重命名)
GET /deck/{deck_id}/cards
GET /card/{card_id}/preview
```

**工作量**: 2天

---

### v1.3 - 用户系统 (优先级: 中)

**目标**: 多用户支持，数据隔离

**功能点**:
- [ ] 用户注册/登录 (邮箱+密码)
- [ ] JWT Token 认证
- [ ] 用户牌组隔离
- [ ] 用户偏好设置 (默认语言、暗色模式)

**数据库变更**:
```sql
CREATE TABLE user (
    id TEXT PRIMARY KEY,
    email TEXT UNIQUE NOT NULL,
    password_hash TEXT NOT NULL,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

ALTER TABLE deck ADD COLUMN user_id TEXT REFERENCES user(id);
```

**工作量**: 3-4天

---

### v1.4 - 邮件通知 (优先级: 中)

**目标**: 处理完成后邮件通知用户

**功能点**:
- [ ] SMTP 配置
- [ ] 邮件模板 (牌组完成通知)
- [ ] 下载链接 (有效期7天)
- [ ] 异步发送

**技术方案**:
- 使用 aiosmtplib 异步发送
- 邮件包含: 标题、卡片数、下载链接

**工作量**: 1-2天

---

### v1.5 - 卡片预览与编辑 (优先级: 中)

**目标**: 导出前可预览和编辑

**功能点**:
- [ ] 卡片列表预览
- [ ] 字幕文本编辑
- [ ] 音频片段试听
- [ ] 删除不需要的卡片
- [ ] 合并/拆分卡片

**技术方案**:
```
GET /deck/{deck_id}/segments
PATCH /segment/{segment_id} (编辑字幕)
DELETE /segment/{segment_id}
POST /segment/{segment_id}/merge (合并相邻)
```

**工作量**: 3天

---

### v1.6 - 批量处理 (优先级: 低)

**目标**: 支持批量提交多个视频

**功能点**:
- [ ] 批量提交 YouTube 链接
- [ ] 批量上传文件
- [ ] 处理队列管理
- [ ] 批量下载

**工作量**: 2天

---

### v1.7 - 分享功能 (优先级: 低)

**目标**: 牌组分享给其他用户

**功能点**:
- [ ] 生成分享链接
- [ ] 分享链接有效期
- [ ] 访问统计
- [ ] 公开牌组库

**工作量**: 2-3天

---

### v1.8 - 更多音频源 (优先级: 低)

**目标**: 支持更多视频/音频平台

**功能点**:
- [ ] Bilibili 支持 (yt-dlp 已支持)
- [ ] 网易云音乐
- [ ] 喜马拉雅
- [ ] 直接音频URL

**工作量**: 2天
